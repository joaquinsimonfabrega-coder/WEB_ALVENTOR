Manual de Uso — Web Alventor

Guía completa para actualizar y mantener la web sin necesidad de conocimientos técnicos avanzados.

1. Panel de Administración

El panel admin te permite añadir, editar y eliminar proyectos y noticias directamente desde el navegador, sin tocar ningún archivo de código.

Cómo acceder

  1. Abre la web con Live Server en VS Code (clic derecho sobre index.html → Open with Live Server).

  2. Ve a la URL: http://127.0.0.1:5500/admin/ — o haz clic en el enlace "Panel Admin" que aparece en el pie de página de cualquier página.

  3. Introduce la contraseña: alventor2024

Importante: Los datos se guardan en el navegador (localStorage). Si borras el historial o cambias de navegador, los datos volverán a los valores por defecto. Para un uso en producción real, consulta la sección de preguntas frecuentes.

En la barra lateral del panel, haz clic en Proyectos. Verás la lista de todos los proyectos actuales.

2. Cambiar Textos de las Páginas

Los textos que no son proyectos ni noticias (títulos de secciones, descripciones, datos de contacto, etc.) están directamente en los archivos .html. Ábrelos en VS Code y edítalos.

Menú de navegación y footerjs/components.js

Cómo encontrar el texto rápidamente

  1. Abre el archivo en VS Code.

  2. Pulsa Ctrl + F para abrir el buscador.

  3. Escribe una parte del texto que quieres cambiar (ej: "Sede Central").

  4. VS Code lo resaltará. Haz clic y edita directamente.

Tip: El título de cada sección empieza siempre con un comentario en mayúsculas como <!-- HERO --> o <!-- CONTACTO PRINCIPAL -->. Usa el buscador para saltar entre secciones.

Datos de contacto (dirección, teléfono, email, horario)

Están en contacto.html, dentro de la sección <!-- CONTACTO PRINCIPAL -->. Busca el texto actual y sustitúyelo.

Textos de la navegación y el footer

Abre js/components.js. Al inicio del archivo verás:

// Links de navegación
const NAV_LINKS = [
  { href: 'quienes-somos.html', label: 'Quiénes Somos' },
  { href: 'servicios.html',    label: 'Servicios' },
  ...
]

Cambia el valor de label para renombrar un elemento del menú.

Para cambiar el texto del footer (dirección, email, copyright, etc.), busca dentro de la función renderFooter() en ese mismo archivo.

3. Cambiar Fotografías

Opción A — Usar fotos propias (recomendado)

  1. Crea una carpeta llamada img dentro de la carpeta WEB ALVENTOR.

  2. Copia tus fotos a esa carpeta. Usa nombres sin espacios ni tildes: obra-murcia.jpg, hero-home.jpg.

  3. En el archivo HTML donde quieres cambiar la foto, localiza el tag <img> y cambia el atributo src:

<!-- Antes -->
<img src="https://lh3.googleusercontent.com/..." alt="...">

<!-- Después -->
<img src="img/mi-foto.jpg" alt="Descripción de la foto">

Opción B — Usar enlace externo

Si la foto está alojada en otra web (Google Drive, Cloudinary, Imgur, etc.), copia el enlace directo a la imagen y pégalo como valor del src.

Atención: No uses el enlace normal de Google Drive. Necesitas el enlace directo al archivo. En Google Drive: clic derecho sobre la imagen → Obtener enlace → cambia la URL a formato directo.

4. Ofertas de Empleo

Las ofertas están en carrera.html, dentro de la sección <!-- POSICIONES ABIERTAS -->. Cada oferta es un bloque de código delimitado por comentarios como <!-- Posición 1 -->.

Editar una oferta existente

  1. Abre carrera.html en VS Code.

  2. Pulsa Ctrl + F y busca el título de la oferta (ej: "Jefe de Obra Internacional").

  3. Edita el título (<h3>), el badge de nivel, la descripción (<p>) y los tres requisitos (<span> con icono).

Añadir una nueva oferta

  1. Copia el bloque completo de una oferta existente (desde <!-- Posición N --> hasta su </div> de cierre).

  2. Pégalo justo antes del </div> que cierra <div class="space-y-5 stagger">.

  3. Modifica el contenido del bloque pegado con los datos de la nueva oferta.

Eliminar una oferta

  1. Localiza el comentario <!-- Posición N --> de la oferta a eliminar.

  2. Selecciona desde ese comentario hasta el </div> de cierre del bloque (el último de la oferta, antes del siguiente comentario <!-- Posición -->).

  3. Pulsa Supr o Backspace.

Cambiar el badge de nivel

El badge de color aparece sobre el título. Hay dos estilos:

<!-- Badge dorado (posición senior/directiva) -->
<span class="... text-[#0b1f3a] bg-[#C49A3C]">Dirección</span>

<!-- Badge azul (posición junior) -->
<span class="... bg-[#30628d] text-white">Junior</span>

Cambia el texto del badge y el color según corresponda.

5. Cambiar Colores y Estilo

Los colores principales están definidos en js/tailwind-config.js. Si cambias un color ahí, se actualiza en toda la web automáticamente.

Archivo: js/tailwind-config.js

colors: {
  primary:  '#000615',   // Negro profundo (fondo hero, texto)
  navy:     '#0b1f3a',   // Azul marino (fondos oscuros, botones)
  steel:    '#30628d',   // Azul medio (iconos, acentos)
  gold:     '#C49A3C',   // Dorado (acento principal, hover)
  bgsoft:   '#fcf8fb',   // Blanco roto (fondo de todas las páginas)
  surface:  '#f6f3f5',   // Gris suave (fondos de secciones)
  border:   '#c4c6ce',   // Gris borde (líneas, separadores)
  muted:    '#75777e',   // Gris texto secundario
  body:     '#44474d',   // Gris texto de párrafos
}

Tip: Para cambiar el color dorado de los acentos por otro color corporativo, cambia únicamente el valor de gold. Afectará a todos los botones hover, barras decorativas y badges dorados de la web.

Animaciones y estilos especiales

Las animaciones (fade-in, transiciones de página, barra de progreso) están en css/custom.css. No es necesario tocarlas salvo que quieras personalizar duraciones o efectos.

6. Menú de Navegación y Footer

El menú y el footer son componentes compartidos que se inyectan en todas las páginas automáticamente desde js/components.js. Cualquier cambio en ese archivo afecta a toda la web.

Cambiar el logo o nombre en el menú

Busca la función renderNav() en js/components.js. Localiza el bloque del logo:

<span class="font-black...">ALVENTOR</span>
<span class="font-light...">EPC &amp; Construcción</span>

Sustituye el texto por el que quieras.

Añadir o quitar páginas del menú

Busca el array NAV_LINKS al inicio de js/components.js y añade o elimina entradas:

{ href: 'nueva-pagina.html', label: 'Nueva Página' }

Cambiar datos del footer

Busca la función renderFooter() en js/components.js. Localiza los textos de dirección, teléfono y email y edítalos directamente.

9. Preguntas Frecuentes

¿Los cambios del panel admin se guardan para siempre?

Se guardan en el localStorage del navegador, que es permanente mientras no borres el historial. Si en el futuro publicas la web en un servidor, necesitarás una base de datos real (se puede añadir). Para uso interno o durante el desarrollo, localStorage funciona perfectamente.

Borré el historial del navegador y perdí los datos. ¿Qué hago?

Los datos por defecto (los ejemplos iniciales) están en js/data.js en los arrays defaultProjects y defaultNews. Cuando el localStorage está vacío, la web los carga automáticamente. Para recuperar tus datos personalizados, tendrás que volver a introducirlos desde el panel admin.

Recomendación: Si añades muchos proyectos y noticias desde el panel, exporta los datos periódicamente. Abre la consola del navegador (F12 → pestaña Console) y escribe localStorage.getItem('alventor_projects') para copiar los datos.

¿Cómo añado una página nueva?

  1. Copia cualquier página existente (ej: contacto.html) y renómbrala (ej: certificaciones.html).

  2. Edita el contenido dentro de <main>. El nav y footer se inyectan solos.

  3. Añade el nuevo enlace en js/components.js dentro del array NAV_LINKS.

¿Cómo publico la web en internet?

La web es 100% estática (solo HTML, CSS y JS), así que puedes subirla a cualquier servicio de hosting gratuito:

  • Netlify — arrastra la carpeta WEB ALVENTOR a netlify.com/drop
  • Vercel — conecta tu carpeta o repositorio de GitHub
  • GitHub Pages — sube los archivos a un repositorio público

¿Por qué no se ven las animaciones al abrir el .html directamente?

Algunos navegadores bloquean scripts cuando el archivo se abre directamente desde el disco (file://). Usa siempre Live Server en VS Code para ver la web correctamente durante el desarrollo.

¿Cómo cambio la contraseña del panel admin?

Abre admin/admin.js y cambia el valor en la primera línea:

const ADMIN_PASSWORD = 'alventor2024'; // ← cambia esto

Nota: Esta contraseña es visible en el código fuente. Para un panel admin en producción real, la autenticación debe hacerse en el servidor, no en el cliente.
